Edit the contact section in `index.html`:

### Customizing Colors
Modify CSS custom properties in `styles.css`:
```css
:root {
    --accent-primary: #00d4ff;
    --accent-secondary: #ff6b6b;
    /* Add your custom colors */
}
```
